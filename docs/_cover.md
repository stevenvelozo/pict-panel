# Pict Panel <small>1</small>

> A developer panel for inspecting and debugging Pict applications at runtime

- **Hotloadable** -- Inject into any running Pict app with a single console command
- **Live Inspection** -- Browse AppData, templates, views, and providers in real time
- **Template Editing** -- Edit templates in-place with persistent override management
- **Export** -- Download your template overrides as JSON or JS source
